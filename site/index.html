AI Agent Programming

Build Real AI Agents with LangChain and LangGraph

A practical, module-based course to design, implement, and evaluate LLM applications and agentic systems, from first chains to advanced RAG and production-ready graph architectures.

Modules

Progressive path based on the O'Reilly book "AI Agents and Applications".

  • M1 Getting Started with LLMs and LangChain Models, prompts, chains, output parsing, and first hands-on lab.
  • M2 Summarization with LangChain and LangGraph Token-aware summarization pipelines and graph-based workflows.
  • M3 RAG Systems Document ingestion, embeddings, retrieval, and grounded generation.
  • M4 Advanced RAG Reranking, query expansion, hybrid retrieval, and evaluation design.
  • M5 AI Agents Architectures with LangGraph State machines, tool orchestration, memory, control, and safety rails.

Course outputs

  • One interactive module page per topic.
  • Guided labs with progressively complex assignments.
  • Exercise sets and implementation deliverables.

Sitography

  • Building Reliable Agents - Course Introduction
    Introductory lesson on designing reliable agent behavior, emphasizing control flow, robustness, and practical agent engineering patterns.
  • Foundation: Introduction to LangChain (Python) - Course Overview
    Foundational overview of LangChain in Python, including core abstractions for prompts, models, chains, and basic application composition.
  • Quickstart: LangGraph Essentials (TypeScript) - Introduction
    Fast entry point to LangGraph concepts in TypeScript, focused on graph-based orchestration for multi-step agent workflows.
  • LangGraph Essentials (Python) - Introduction
    Introduction to LangGraph in Python, covering stateful execution, node/edge modeling, and practical patterns for agent pipelines.
  • ADK - Get Started
    Quick-start documentation for ADK, useful for understanding agent development workflows, project setup, and implementation fundamentals.
  • Building LLM Applications (GitHub Repository)
    Source repository for practical labs and exercises used in module M7, with runnable examples and implementation-oriented assignments.
