# Agents Course

Interactive course website and teaching materials for:

**AI Agent Programming (LangChain + LangGraph)**

## Run locally

From project root:

```bash
cd site
python3 -m http.server 8080
```

Open: `http://localhost:8080`

## Language switch

The website is authored in English. The `EN/IT` selector applies Google Translate on-page to Italian when selected.

## Modules

- M1: Getting Started with LLMs and LangChain
- M2: Summarization with LangChain and LangGraph
- M3: RAG Systems
- M4: Advanced RAG
- M5: AI Agents Architectures with LangGraph
